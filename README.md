# DigiTools

A digital tools buying web app - providing essential packages to help you stay up to date with the latest productive tools.

### ⚙ Tech Stack

- React.js
- TailwindCSS, DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON (for data)

### 🌟 Key Features

- One/Multiple products can be added to cart
- Specific product/s can be removed or checked out from the cart
- Smooth and accessible UI/UX
